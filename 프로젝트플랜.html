무방 앱 개발 프로젝트 계획

프로젝트 개요

골프 스윙 연습기와 연동되는 안드로이드 앱 개발

  • 실시간 스윙 데이터 분석 및 피드백 제공
  • 초보자와 중급자 모두 사용 가능한 직관적 UI/UX

개발 환경 및 기술 스택

  • 개발 IDE: Android Studio
  • 개발 언어: Kotlin
  • 에뮬레이터: Android Studio AVD
  • 협업 도구: 닷홈 웹 호스팅, FTP, (향후) Git 등

앱 주요 기능

  1. 스윙 속도 측정 및 구간별 LED 색상 표시
  2. 스윙 궤적 분석 및 개선점 피드백
  3. 일관된 스윙 궤도 학습 가이드
  4. 연습 결과 누적 및 통계 제공

개발 절차 및 일정

  1. 요구사항 정의 및 기능 명세 (1주)
  2. 아키텍처 설계 및 주요 컴포넌트 도출 (1주)
  3. 개발 환경 셋업 및 기술 검증 (1주)
  4. 기능 개발 및 테스트 (6주)
    • BLE 통신 모듈
    • 실시간 데이터 처리/분석
    • UI/UX 개발
  5. 통합 테스트 및 이슈 수정 (2주)
  6. 베타 테스트 및 피드백 반영 (2주)
  7. 스토어 배포 준비 및 런칭 (1주)
