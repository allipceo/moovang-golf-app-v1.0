FAQ 주요 질문과 답변

🔗 연결 관련 FAQ

  • Q: 블루투스가 연결되지 않아요
    • A: 센서 전원 확인 → 스마트폰 블루투스 재시작 → 앱 재실행 순으로 시도하세요
  • Q: 연결이 자주 끊어져요
    • A: 센서와 폰 사이 거리를 2m 이내로 유지하고, 다른 블루투스 기기와의 간섭을 피하세요
  • Q: 여러 센서를 동시에 사용할 수 있나요?
    • A: 현재 버전에서는 한 번에 하나의 센서만 연결 가능합니다

📊 측정 관련 FAQ

  • Q: 측정 결과가 부정확한 것 같아요
    • A: 연습기를 실제 골프채처럼 자연스럽게 스윙하세요. 무리한 힘은 피하세요
  • Q: 20회 측정 중간에 멈출 수 있나요?
    • A: 네, 일시정지 버튼으로 언제든 중단하고 재개할 수 있습니다
  • Q: 실내와 실외 측정 차이가 있나요?
    • A: 측정값은 동일하지만, 실외에서 더 자연스러운 스윙이 가능합니다

📈 분석 관련 FAQ

  • Q: 스윙 궤도 색깔의 의미는 무엇인가요?
    • A: 초록색(표준), 주황색(인사이드아웃), 파란색(아웃사이드인) 등으로 궤도 타입을 구분합니다
  • Q: 목표 속도는 어떻게 설정하나요?
    • A: 설정 > 개인정보에서 현재 실력보다 5-10% 높게 설정하는 것을 권장합니다
  • Q: 데이터를 다른 기기로 옮길 수 있나요?
    • A: 설정 > 데이터 관리에서 백업 파일을 생성하여 다른 기기로 이동 가능합니다

검색 기능 구현

function searchHelp(keyword) {
    const searchTerm = keyword.toLowerCase().trim();
    const faqItems = document.querySelectorAll('.faq-item');
    const quickStartSteps = document.querySelectorAll('.quick-start-step');
    
    if (searchTerm === '') {
        // 모든 항목 표시
        faqItems.forEach(item => item.style.display = 'block');
        quickStartSteps.forEach(step => step.style.display = 'block');
        return;
    }
    
    // FAQ 검색
    faqItems.forEach(item => {
        const question = item.querySelector('.question').textContent.toLowerCase();
        const answer = item.querySelector('.answer').textContent.toLowerCase();
        
        if (question.includes(searchTerm) || answer.includes(searchTerm)) {
            item.style.display = 'block';
            highlightResults(item, searchTerm);
        } else {
            item.style.display = 'none';
        }
    });
    
    // 빠른 시작 가이드 검색
    quickStartSteps.forEach(step => {
        const content = step.textContent.toLowerCase();
        step.style.display = content.includes(searchTerm) ? 'block' : 'none';
    });
}

function contactSupport() {

안녕하세요. 무방 앱 사용 중 문의사항이 있습니다.

[문의 내용을 여기에 작성해주세요]

- 앱 버전: ${appVersion}
- 기기 정보: ${userAgent}

    `.trim();

}
