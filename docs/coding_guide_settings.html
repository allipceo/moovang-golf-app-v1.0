🎯 페이지 핵심 기능

1. 블루투스 연결 관리

  • 연결 상태 표시: 현재 연결된 기기 정보
  • 기기 검색: 사용 가능한 무방 센서 목록
  • 연결/해제: 버튼으로 간편 조작
  • 신호 강도: 연결 품질 시각화
  • 재연결: 연결 실패시 자동 재시도

2. 개인정보 입력

  • 기본 정보: 이름, 성별, 나이
  • 골프 정보: 핸디캡, 주 사용 클럽
  • 목표 설정: 목표 스윙속도, 연습 횟수
  • 저장/불러오기: 입력 데이터 영구 보관
  • 초기화: 설정 리셋 기능

3. 프로그램 정보

  • 앱 정보: 버전, 업데이트 날짜
  • 사용법: 간단한 가이드
  • 센서 정보: 하드웨어 사양
  • 문의처: 고객지원 연락처
  • 라이선스: 오픈소스 정보

📡 블루투스 설정 섹션

  • 현재 상태: 연결됨/미연결 표시
  • 기기 목록: "무방 센서 01", "무방 센서 02" 등
  • 연결 버튼: 큰 버튼으로 쉽게 접근

👤 개인정보 섹션

  • 이름: 텍스트 입력 (2자 이상)

  • 나이: 숫자 입력 (10-99세)

  • 목표 속도: 숫자 입력 + 슬라이더
  • 주간 연습: 횟수 선택 (1-7회)
  • 알림 설정: 토글 스위치

3. 하단 영역

  • 저장 버튼: 전체 설정 저장
  • 초기화 버튼: 기본값으로 복원
  • 내보내기 버튼: 설정 백업 (추후 확장)

검증 함수 예시

function validateName(name) {
    if (name.length < 2) {
        return { valid: false, message: "이름은 2자 이상 입력해주세요" };
    }
    const regex = /^[가-힣a-zA-Z0-9\s]+$/;
    if (!regex.test(name)) {
        return { valid: false, message: "올바른 이름을 입력해주세요" };
    }
    return { valid: true };
}

function validateAge(age) {
    const ageNum = parseInt(age);
    if (isNaN(ageNum) || ageNum < 10 || ageNum > 99) {
        return { valid: false, message: "나이는 10-99 사이로 입력해주세요" };
    }
    return { valid: true };
}

🎯 블루투스 시뮬레이션

기기 검색 및 연결 시뮬레이션

// 가상 기기 목록
const availableDevices = [
    { name: "무방 센서 01", signal: 5, battery: 85 },
    { name: "무방 센서 02", signal: 3, battery: 92 },
    { name: "무방 센서 03", signal: 2, battery: 67 }
];

function simulateDeviceScan() {
    return new Promise((resolve) => {
        setTimeout(() => {
            const foundDevices = availableDevices.filter(
                () => Math.random() > 0.3 // 70% 확률로 발견
            );
            resolve(foundDevices);
        }, 2000); // 2초 검색 시간
    });
}

function simulateConnection(deviceName) {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            if (Math.random() > 0.1) { // 90% 성공률
                resolve({
                    success: true,
                    device: deviceName,
                    signal: Math.floor(Math.random() * 5) + 1
                });
            } else {
                reject(new Error("연결에 실패했습니다"));
            }
        }, 3000); // 3초 연결 시간
    });
}
