비거리(야드) = 헤드스피드(m/s) × 5.5 + 랜덤팩터(-10~+15)

🎨 궤도 타입별 시각적 특성

1. 표준 궤도

• 색상: 녹색(백스윙) → 노란색(다운스윙) → 빨간색(임팩트)

• 형태: 적당한 호를 그리며 자연스러운 궤도

2. 인사이드아웃 궤도

• 색상: 연두색(백스윙) → 주황색(다운스윙) → 빨간색(임팩트)

• 형태: 안쪽에서 바깥쪽으로 향하는 이상적 궤도

3. 아웃사이드인 궤도

• 색상: 파란색(백스윙) → 보라색(다운스윙) → 빨간색(임팩트)

• 형태: 바깥쪽에서 안쪽으로 향하는 슬라이스 궤도

4. 업라이트 궤도

• 색상: 하늘색(백스윙) → 청록색(다운스윙) → 빨간색(임팩트)

• 형태: 수직에 가까운 가파른 궤도

5. 플랫 궤도

• 색상: 갈색(백스윙) → 황토색(다운스윙) → 빨간색(임팩트)

• 형태: 수평에 가까운 평면적 궤도
