🏌️ 무방(MOOVANG) V1.0 개발계획서

골프 스윙 연습기 앱 개발 프로젝트

작성일: [2025.01.10 13:45] | 작성자: 노팀장

📋 1. 프로젝트 개요

1.1 프로젝트 목표

  • 핵심 목표: LED 및 자이로스코프를 활용한 골프 스윙 연습기와 연동되는 모바일 앱 개발
  • 주요 기능: 스윙 속도 측정, 스윙 궤도 분석, 임팩트 지점 최적화
  • 사용자 가치: 비거리 향상, 방향성 개선, 즐거운 골프 연습 경험

1.2 개발 범위

  • 1차 목표: 안드로이드/iOS 모바일 앱
  • 2차 목표: PC용 앱 (추후 확장)
  • 3차 목표: 웹 기반 분석 도구

🎯 2. 핵심 기능 정의

2.1 속도 측정 서비스

  • 스윙 전 과정에서 구간별 스윙 스피드 측정
  • 속도별 색상 변동 (5단계: 빨강→노랑→초록→파랑→보라)
  • 최고 스피드와 스피드 구간 실시간 인식

2.2 임팩트 존 서비스

  • 자이로 센서 기반 최저점 감지
  • 임팩트 지점에서 음향 피드백
  • 최고 스피드와 임팩트 지점 싱크 여부 확인

2.3 스윙 궤도 서비스

  • 3차원 스윙 궤도 추적 및 시각화
  • 기준 궤도 설정 및 비교 분석
  • 개인별 최적 스윙 궤도 학습

2.4 데이터 관리 서비스

  • 블루투스 연동을 통한 실시간 데이터 수신
  • 연습 기록 누적 및 히스토리 관리
  • 맞춤형 코칭 데이터 제공

🛠️ 3. 기술 스택

구분 기술 선택 이유
개발 플랫폼 React Native 크로스 플랫폼 지원, 단일 코드베이스
데이터베이스 SQLite (로컬) + Firebase (클라우드) 오프라인 지원 + 클라우드 백업
블루투스 통신 React Native Bluetooth 하드웨어 센서 연동
차트/그래프 React Native Chart Kit 스윙 데이터 시각화
3D 렌더링 Three.js 스윙 궤도 3차원 표현

📅 4. 개발 일정

Phase 1: 기획 및 설계 (1-2주)

  • 요구사항 분석 완료
  • UI/UX 설계
  • 시스템 아키텍처 설계
  • 개발 환경 구축

Phase 2: 핵심 모듈 개발 (3-4주)

  • 블루투스 연동 모듈
  • 데이터 수신 및 처리 모듈
  • 기본 UI 컴포넌트
  • 데이터베이스 구조

Phase 3: 기능 구현 (4-5주)

  • 스윙 측정 화면
  • 데이터 분석 화면
  • 히스토리 관리
  • 설정 관리

Phase 4: 테스트 및 배포 (2주)

  • 단위 테스트
  • 통합 테스트
  • 사용자 테스트
  • 앱스토어 배포

📱 5. UI/UX 설계 방향

5.1 화면 구성

  • 초기화면: 앱 로딩 및 블루투스 연결 상태
  • 메뉴화면: 환경설정, 스윙측정, 스윙분석 메뉴
  • 스윙측정: 실시간 측정 및 결과 표시
  • 스윙분석: 금일분석, 과거분석 데이터 시각화
  • 환경설정: 블루투스 연결, 개인정보, 프로그램 설명

5.2 디자인 원칙

  • 직관적이고 간단한 인터페이스
  • 골프 테마 색상 (그린, 화이트, 골드)
  • 대형 버튼으로 야외 사용성 확보
  • 실시간 피드백 중심의 UX

🎮 6. 개발 모듈 구조

모듈명 주요 기능 우선순위
BluetoothManager 하드웨어 연결 및 데이터 수신 높음

ReportGenerator 분석 리포트 생성 낮음

✅ 7. 검증 계획

7.1 단계별 검증

  • 모듈 검증: 각 모듈별 단위 테스트
  • 통합 검증: 모듈 간 연동 테스트
  • 성능 검증: 실시간 데이터 처리 속도
  • 사용성 검증: 실제 골퍼 테스트

7.2 품질 기준

  • 블루투스 연결 성공률 95% 이상
  • 데이터 처리 지연시간 100ms 이하
  • 앱 크래시율 1% 이하
  • 사용자 만족도 4.0/5.0 이상
